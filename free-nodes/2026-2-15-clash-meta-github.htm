什么是Shadowrocket？

Shadowrocket是一款专为iOS平台设计的网络代理工具，它支持多种协议，包括Shadowsocks、Vmess、Trojan等。它能够通过代理服务器绕过网络限制，使用户能够访问被封锁的网站和应用，享受更自由的互联网体验。Shadowrocket不仅适用于浏览器，还支持各种应用程序的网络流量，使其成为一款功能全面且实用的工具。

Shadowrocket的主要功能

1. 多协议支持

Shadowrocket支持多种代理协议，包括Shadowsocks、Vmess、Trojan等。这些协议的支持使得用户可以根据自己的需要选择适合的方案，从而实现更好的上网体验。例如，Shadowsocks通常以其简单、高效的特点受到广泛欢迎，而Vmess和Trojan协议则适用于更加注重隐私保护和安全性的用户。

2. 用户友好的界面

Shadowrocket的界面设计简单直观，用户可以轻松地进行配置和管理。无论是新手还是有经验的用户，都能够快速上手，并通过应用内提供的各项功能实现对代理设置的精准控制。

3. 自定义配置

Shadowrocket允许用户自定义代理配置，用户可以根据自己的需求设置不同的节点、端口、加密方式等，甚至可以指定特定的应用使用不同的代理服务器。这种灵活性使得Shadowrocket能够满足各种不同的使用场景。

4. 流量统计

作为一款网络代理工具，Shadowrocket提供实时流量统计功能。通过流量统计，用户可以清楚地了解自己在使用代理服务过程中的数据消耗情况，帮助合理管理网络资源，避免不必要的数据浪费。

Shadowrocket的配置教程

配置Shadowrocket的过程非常简单，但需要准确输入代理服务器的信息。以下是详细的配置步骤。

添加代理服务器

  1. 打开Shadowrocket应用：在主界面中，点击右上角的“+”号，进入配置页面。

  2. 选择添加配置：在弹出的菜单中，选择“添加配置”选项。

  3. 输入代理信息：根据您所使用的代理服务提供商的要求，输入服务器的地址、端口、密码等相关信息。确保这些信息准确无误。

  4. 保存配置：输入完毕后，点击右上角的“保存”按钮，完成配置。

选择代理模式

Shadowrocket提供三种代理模式，用户可以根据自己的需求选择合适的模式：

  1. 全局模式：所有流量都通过代理服务器，无论您访问哪个网站或使用哪个应用，都会经过代理。适用于需要全程匿名访问互联网的场景。

  2. 分应用模式：只有指定的应用程序流量通过代理服务器，其它应用则正常访问互联网。适用于需要在某些应用中使用代理的场景。

  3. 手动模式：用户可以手动选择需要通过代理访问的应用程序。适合用户灵活控制不同应用的代理使用。

测试连接

配置完成后，返回到主界面，点击您刚刚添加的代理配置，然后选择“连接”。如果连接成功，您会看到连接状态显示为“已连接”，这时就可以开始正常使用代理服务了。

FAQ

为什么使用 CDN 后仍然能被封？如何规避？
若 CDN 回源 IP 或配置不当可能泄露真实服务器信息，或者 CDN 本身被目标防火墙识别。应启用完整的反向代理配置、隐藏源站 IP 与禁用不必要的服务，必要时使用多级中转或更换 CDN 供应商。
VMess 使用 WebSocket 是否有安全优势？
WebSocket 可伪装成普通 HTTP/HTTPS 流量，配合 TLS 加密，可隐藏真实通信特征，提升抗封锁能力。结合策略组使用，可保证访问速度和网络稳定性，同时增加节点安全性。
Netch 如何结合节点测速实现自动分流？
Netch 可以结合节点延迟检测和策略组，将不同类型的流量分配到低延迟节点。提高网页、视频和游戏访问速度及稳定性，优化整体网络体验。
Netch 无法识别某游戏流量时应如何处理？
请确认已启用 TUN 模式并在游戏路径中选择正确的执行文件。部分游戏使用反作弊系统需管理员权限运行 Netch；若仍无法识别，可尝试手动添加端口规则。
SSR 协议的多路复用功能如何提升性能？
SSR 多路复用允许多个短连接共享同一通道，减少握手次数和加密开销。适合游戏、即时通讯等高频短连接场景，提升网络吞吐量和连接稳定性，提高整体上网体验。
Clash 运行在低配设备上卡顿，如何减轻资源占用？
减少 rule-provider 数量、关闭不必要的日志与统计、降低 DNS 轮询频率并禁用内置测速；若使用 TUN 模式，可改为代理模式减少内核负担。必要时升级到更高性能的设备或使用更轻量内核。
PassWall2 如何实现智能分流？
PassWall2 通过自定义规则文件控制流量走向。可根据目标域名、国家或 IP 范围自动判断走代理或直连。用户还能加载社区规则，实现类似 Clash 的高效路由分流。
Potatso Lite 显示证书错误，如何修复？
证书错误通常由域名与证书不匹配或证书链不完整引起。请检查服务器证书是否为有效域名签发并包含完整链，如果使用 Let's Encrypt，请确保证书未过期并已正确安装到反向代理或服务端。
Clash 的延迟检测如何帮助节点选择？
Clash 会定期测速节点延迟，并根据策略组自动选择延迟最低节点处理流量。可避免访问高延迟或失效节点，提高网页、视频和游戏访问速度及稳定性。
Matsuri 的节点自动测速功能如何配置？
Matsuri 可以设置测速周期和节点列表，程序会自动测试延迟和速度并排序。用户可根据测试结果选择最佳节点，提高网络访问速度和使用体验。
